# Contributing to MagnetarHeimarmene

## Getting Started

1.  **Clone** the repository.
2.  **Install** dependencies: `kdepim-runtime`, `kaccounts-integration`, Plasma 6 SDK.
3.  **Run** the widget with `plasmoidviewer -a .`.

## Process

1.  Pick a `ready` task from `PLAN.md`.
2.  Update task state to `in_progress`.
3.  Create a branch.
4.  Work & Commit.
5.  Create PR.
